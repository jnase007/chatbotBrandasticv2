Step 1: Upload Files to Your Server

Upload these files to your brandastic.com server:

  • Backend API: Upload the entire server/ folder to /chatbot-api/
  • Embed Script: Upload public/embed.js to your website's public directory

Step 2: Install & Start the API

SSH into your server and run:

cd /path/to/chatbot-api npm install node index.js

The API will run on port 3001 and handle all chatbot requests.

Step 3: Add One Line to Your Website

Add this single line to your website footer (before closing </body> tag):

<script src="https://brandastic.com/embed.js" async></script>
