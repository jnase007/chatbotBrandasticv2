üîß WordPress Integration Methods

Method 1: Theme Footer (Recommended)

Add to your theme's footer.php file before </body>:

<!-- Brandastic Chatbot -->
<script>
(function() {
  var script = document.createElement('script');
  script.src = 'https://your-chatbot-domain.com/embed.js';
  script.async = true;
  document.head.appendChild(script);
})();
</script>

Method 2: WordPress Functions.php

Add this to your theme's functions.php file:

// Add Brandastic Chatbot
function add_brandastic_chatbot() {
    ?>
    <script src="https://your-chatbot-domain.com/embed.js" async></script>
    <?php
}
add_action('wp_footer', 'add_brandastic_chatbot');

Method 3: WordPress Plugin

Create a simple plugin file brandastic-chatbot.php:

<?php
/*
Plugin Name: Brandastic Chatbot
Description: Adds the Brandastic AI chatbot to your website
Version: 1.0
*/

function brandastic_chatbot_script() {
    wp_enqueue_script(
        'brandastic-chatbot',
        'https://your-chatbot-domain.com/embed.js',
        array(),
        '1.0',
        true
    );
}
add_action('wp_enqueue_scripts', 'brandastic_chatbot_script');
